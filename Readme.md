# 🌸 Iris Flower Classification - CODSOFT Internship

## 📌 Project Overview

This project focuses on building a Machine Learning model to classify Iris flowers into three different species:

* Setosa
* Versicolor
* Virginica

The classification is performed using flower measurements such as sepal length, sepal width, petal length, and petal width.

This project was completed as part of the **Data Science Internship at CODSOFT**.

---

## 🎯 Objective

The objective of this project is to:

* Understand and explore the Iris dataset
* Perform data preprocessing and visualization
* Train a classification model
* Evaluate the performance of the model
* Gain hands-on experience with Machine Learning workflow

---

## 📂 Dataset Information

The dataset contains the following features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width
* Species (Target Variable)

Each row represents one Iris flower sample.

## 🔍 Project Workflow

### 1️⃣ Data Loading

* Loaded the dataset using Pandas.

### 2️⃣ Data Cleaning

* Removed unnecessary columns (if any).
* Checked for missing values.

### 3️⃣ Exploratory Data Analysis (EDA)

* Visualized class distribution.
* Used pairplots to analyze feature relationships.

### 4️⃣ Data Splitting

* Split the dataset into training and testing sets (80:20 ratio).

### 5️⃣ Model Training

* Applied Logistic Regression classifier.

### 6️⃣ Model Evaluation

* Evaluated using:

  * Accuracy Score
  * Classification Report
  * Confusion Matrix

---

## 🤖 Model Used

**Logistic Regression**

Logistic Regression is a classification algorithm that predicts the probability of a data point belonging to a particular class.

---

## 📊 Results

* The model achieved approximately **97–100% accuracy** on the test data.
* The confusion matrix showed very few misclassifications.
* Petal length and petal width were strong indicators for species classification.

---

## 📈 Conclusion

The Logistic Regression model successfully classified Iris flowers with high accuracy.
